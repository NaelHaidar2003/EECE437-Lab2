# EECE437-Lab2

# Project Setup

## Clone the Repository
```sh
git clone https://github.com/NaelHaidar2003/EECE437-Lab2.git
```

## Navigate to the Project Folder
```sh
cd Lab2
```

## Restore Dependencies
```sh
dotnet restore
```

## Build the Project
```sh
dotnet build
```

## Run the Application
```sh
dotnet run
```
